# Documentation
## Added parameters
### Grouping by position 
``--num_groups <number of groups>``  
0=random
1=no groups  

``--group_by_postition <1 or 0>``  
activate grouping by dingsda

``--margin_groups <minimum margin between the center of the groups>``

### Grouping by similarity
``--group_by_similarity <1>``  
activate group by similarity. All are same except one attribute
